In the name of the so-called "Good Programming", I would like to note one
thing regarding your StrComp example. You have written:

if strcomp(s1, s2, 0)=0 then
return true
else
return false
end if

But this can be written like:

return strComp(s1,s1,0) = 0

Which would be an easier solution. This way of writing it is common in C and Pascal.
You can use '<' and '>' in the same way; to make a function returning
whether s1 is bigger that s2, use:

return len(s1) > len(s2)
