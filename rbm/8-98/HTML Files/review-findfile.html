RB Find File
Author: James Milne (Essence Software)
Price: free
Download: Realbasic café (Hotline)

  RB Find File is a plugin that does as its name implies: it finds files. After you install it into the plugins folder, a new control appears in the toolbar–a find file control. You can tell this method to find a file, but you can only search by "name contains ___", a major drawback. To test its accuracy, I searched for "realbasic" on my hard drive. Here were the results:

Results for "realbasic" in example:
Items Found: 56
Items that existed: 2
Number of duplicates: 11

  As you may have guessed, I was shocked! You can expect not to find any duplicates, and certainly not any non-existing folders. Worst of all, the two items that didn't exist were duplicates!

  James Milne claims that his plugin is faster than the Find File app, and I most certainly agree. However, it's not THAT much faster, and considering that the find file program can search by 12 different criterion, and can even search by a file's contents, I'm not surprised. And, to top all this off, it doesn't even have a Read Me file!

  RB Find File makes finding files easy and fast, but that's about all you can say about it that's good. It's output is pathetic–it returns duplicates and non-existent folders! RB Find File is a plugin to be avoided until James Milne makes it work properly.

 Pros: Makes searching for files quick and easy.
 Cons: Outputs large numbers of duplicates and non-existing files (i.e. they end in "::"). Can only search by name. No documentation.
Rating: 1.5
