News

Realbasic goes commercial

   As was guaranteed last issue, REALbasic went commercial in early July. It costs $99.95, or $59.95 if you're under 18. You can also get the 600-page, printed documentation for just under $40, and a CD full of RB stuff for $10. This pricing system makes it the cheapest programming language you can get for the mac—CodeWarrior is $400!

RB gets praise from the press

   Following its Macworld New York debut, Realbasic received rave reviews all over the place! It's the only powerful rapid-development program for the mac (in other languages, you create controls by using commands, not by drawing them), which makes it extremely useful. As Andy Inahtko commented, this makes it easy to concentrate on design, and not so much on the actual programming. Also, MacAddict has made RB the official programming language on their CD. What this means, I'm not exactly sure...

Anti-Aliasing for RB

   A new thread this month allows RB programs to anti-alias pictures. Though it is extremely slow (about 12 seconds for a 50x50 8-bit picture on a 200Mhz 603e!), it is also very good. Also, it is a thread, which allows it to run in the background. You can get this thread from the Realbasic café.
   Curious about this, I made my own, which is much faster. It also gives you the option of either blurring or smoothing the graphic. It too is available at the Realbasic café.

Eject that disk, and stuff it!

   James Milne has written several small plugins this month that do various things. One, EjectDisk, adds one method to RB: Eject(vol as folderitem). This ejects the folderitem (if its possible). The other, StuffPlug, does as its name implies–it uses StuffIt. It adds Stuff and UnStuff methods, which allow you to use StuffIt in conjunction with your RB programs.

First versions of DR2 released

   As was promised, the first versions of DR2 have been released soon after RB went commercial. Though the first three revisions of DR2 aren't official and don't have release notes, they do exist. If you look for new features in the reference, then you're out of luck. They're not there. Instead, just look at this file for all the release notes.
Serial and Sockets revamped

   In the first two revisions of DR2, there were properties added to the socket and serial controls. As you can see from the nifty chart to the right, the PPPStatus (integer) property of a socket allows you to get all the PPP info that you need. Socket controls also have PPPConnect and PPPDisconnect methods now.
   Sockets weren't the only communications control that got an improvement though. A new SerialPort class was added, which allows you to get the name of the port, its input and output drivers, as well as its rated and maximum speeds.
   There are two ways you can get a serial port class. Serial controls now have a .SerialPort property, which gives you access to their ports. Also, the new system object has two properties: SerialPortCount, and an array of all the serial ports.
