First let me say congratulations on getting our first issue out. It looks great and I really like the popup menu at the top. Now let me clear up a few things:

1) CR1 and 1.0.0 are the same thing. CR1 means Commercial Release 1. It will have (as you already know) the VB Importer but no Java support. We will add the Java Compiler back in DR2 once we have done more work on it. DR2 should be available to registered users shortly after the release of CR1.

2) The problems you had with the VB Importer may have been a result of attempting to import a non-VB5 form. We are not guaranteeing support for VB versions prior to 5.0. I would appreciate it if you could try a VB5 file and then update your page with the results of your experience.

That said, thanks for creating what I am sure will be a monthly visit for REALbasic users worldwide.
