Last Words

I met up with Geoff Pearlmen on Hotline towards the start of the month, and he kindly gave me some very old release notes. Here they are, the first in a three-part series of old release notes. For the first time, the release notes for DR1r4-6:

DR1r6 (28 June 1997)
Added DoModal method for windows
Fixed crashing bug with edit fields on 68k Macs
Extended import of graphics to allow files with .jpg and .gif extensions (regardless of Mac file type)
Added chapter 12 (Windows) to documentation
DR1r5
Added rnd, ceil, floor, round, abs, max, min
Added sound support (for 'sfil' sound files)
Improved memory management
DR1r4
Added various higher order maths functions
