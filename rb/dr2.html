Features added in DR2r29 were:
- CellType(Row As Integer, Column As Integer) As Integer (Read/write)
- ColumnType(Column As Integer) As Integer (Read/Write)
The values for these properties are:
0 - default
1 - normal
2 - add checkbox
3 - inline editable
The actual type of a cell depends on the cellType() of the cell - if
default, then the columnType().
eg the columnType might be 2, but if cellType() = 1 then the cell will be
normal.

get/set the checked status of a cell

Warning about DR2s
A few things to note about the DR2 alphas:
* nothing is guaranteed to remain the same - though I'll try and avoid
arbitrarily changing things
* DR2 is the next major version of REALbasic, due to be released at the
end
of the year, at which point you'll no longer be able to use DR2 without
upgrading - so don't use the features if you don't expect to upgrade.
* For the bleeding edge releases, this is all the official documentation available - though as things solidify, the official documentation will be brought up to date.
